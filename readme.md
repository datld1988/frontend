# NodeJS - Boiler Template

## NodeJS Installation
1. Clone this project
```
git clone http://117.103.207.105/boilerplate/ng-nodejs.git .
```

2. Remove ```.git``` folder
```
rm -rf .git
```

3. Add new remote repository
```
git remote add origin {YOUR_GIT_REPOSITORY_HERE}
```

4. Install dependencies
```
npm install
```

5. Enjoy!
```
npm start
```
